Este repositorio implementa una API BackEnd con operaciones CRUD (Crear, Leer, Actualizar, Eliminar). El proyecto está desarrollado principalmente en TypeScript y utiliza CSS para estilos, además de tener algunos archivos en otros lenguajes.

Tecnologías principales
TypeScript (70.9%)
CSS (24.8%)
Otros lenguajes menores: Bru, archivos varios.
Características
Operaciones básicas CRUD para recursos backend.
Código limpio y organizado en TypeScript.
Estructura modular y fácil mantenimiento.
Archivos de estilos con CSS.
Listo para ser utilizado como plantilla para nuevos proyectos o ampliaciones.
Instalación
Clona este repositorio:

bash
git clone https://github.com/jeanneryssmoreno/backend-crud.git
cd backend-crud
Instala las dependencias:

bash
npm install
Configura las variables de entorno si es necesario.

Corre el proyecto:

bash
npm start
Uso
Haz solicitudes HTTP (GET, POST, PUT, DELETE) a los endpoints provistos para gestionar la información a través de la API.
